1. Ma trận đa thức đặc trưng (Characteristic Matrix Polynomial)

Từ phương trình cân bằng ở Chương 1, phương pháp spectral expansion đòi hỏi ta giải phương trình:

{% include components/box-start.html type="formula" title='Phương trình đặc trưng' %} \[ Q(x) \boldsymbol{\psi} = \mathbf{0} \]

với ma trận đa thức đặc trưng:

\[ Q(x) = A_2 + A_1 x + A_0 x^2 = \begin{bmatrix} \lambda x^2 - (\lambda + \mu_1)x & \mu_1 x \\ \mu_2 & \lambda x^2 - (\lambda + \mu_2)x \end{bmatrix} \] {% include components/box-end.html %}

Để tồn tại vector riêng \(\boldsymbol{\psi} \neq \mathbf{0}\), ta cần:

\[ \det[Q(x)] = 0 \]

2. Trị riêng (Eigenvalues)

Khai triển định thức:

{% include components/box-start.html type="formula" title='Đa thức đặc trưng (eq 17)' %} \[ \det[Q(x)] = [\lambda x^2 - (\lambda + \mu_1)x][\lambda x^2 - (\lambda + \mu_2)x] - \mu_1 \mu_2 x = 0 \]

Rút gọn (chia cho \(x\), vì \(x = 0\) là nghiệm tầm thường):

\[ \lambda^2 x^3 - \lambda(2\lambda + \mu_1 + \mu_2)x^2 + (\lambda^2 + \lambda\mu_1 + \lambda\mu_2 + \mu_1\mu_2)x - (\lambda + \mu_1)(\lambda + \mu_2) + \mu_1\mu_2 \cdot \frac{1}{1} = 0 \]

Đây là đa thức bậc 4 theo \(x\) (trước khi chia cho \(x\)).

{% include components/box-end.html %}

Bài báo chỉ ra rằng \(x = 0\) là một nghiệm tầm thường. Chia cho \(x\), ta được đa thức bậc 3, nhưng kết hợp với cấu trúc đặc biệt, phương trình ban đầu bậc 4 có thể giải bằng công thức tường minh.

3. Giải phương trình bậc 4 (Quartic Formula)

Bài báo sử dụng kết quả của Borwein & Erdélyi (1995) để tìm công thức tường minh cho 4 nghiệm:

{% include components/box-start.html type="formula" title='Bốn trị riêng r₁, r₂, r₃, r₄ (eqs 18–21)' %}

Đặt các hệ số trung gian:

\[ \alpha = \mu_1 + \mu_2, \quad \beta = \mu_1 \mu_2 \]

Bốn nghiệm của \(\det[Q(x)] = 0\) là:

\(r_1 = 0\) (nghiệm tầm thường).

\(r_2 = 1\) (nghiệm ứng với điều kiện chuẩn hóa).

\(r_3, r_4\): hai nghiệm còn lại, cho bởi công thức tường minh phụ thuộc \(\lambda, \mu_1, \mu_2\).

Cụ thể (từ eqs 18–21 trong bài báo):

\[ r_3 = \frac{2\lambda + \alpha + \sqrt{D}}{2\lambda}, \quad r_4 = \frac{2\lambda + \alpha - \sqrt{D}}{2\lambda} \]

với \(D = \alpha^2 - 4\beta + 4\lambda\alpha\) (biệt thức, luôn dương khi \(\mu_1 \neq \mu_2\)).

Lưu ý: Đây là dạng đơn giản hóa. Công thức chính xác trong bài báo phức tạp hơn và dựa trên quartic formula.

{% include components/box-end.html %}

4. Proposition 2 (Three eigenvalues inside the unit circle)

{% include components/box-start.html type="theorem" title='Proposition 2' %}

Khi điều kiện ổn định \(\lambda < \mu_1\mu_2/(\mu_1 + \mu_2)\) thỏa mãn, có đúng 3 trị riêng nằm trong đường tròn đơn vị (unit circle):

  • \(r_1 = 0\) (hiển nhiên \(|r_1| < 1\)).
  • \(|r_3| < 1\) và \(|r_4| < 1\).

Nghiệm \(r_2 = 1\) nằm trên đường tròn đơn vị, không dùng trong biểu diễn spectral (nhưng cần cho điều kiện chuẩn hóa).

{% include components/box-end.html %}

Tại sao quan trọng? Trong phương pháp spectral expansion, chỉ các trị riêng với \(|x_k| < 1\) mới đóng góp vào nghiệm (để tổng \(\sum a_k x_k^i\) hội tụ khi \(i \to \infty\)). Có 3 trị riêng phù hợp cho ma trận pha kích thước \(2 \times 2\) — vừa đủ để xác định duy nhất nghiệm.

5. Vector riêng (Eigenvectors)

Với mỗi trị riêng \(x_k\) (\(k = 1, 3, 4\)), vector riêng tương ứng \(\boldsymbol{\psi}_k = (\psi_{k,1}, \psi_{k,2})\) thỏa mãn \(Q(x_k)\boldsymbol{\psi}_k = \mathbf{0}\).

{% include components/box-start.html type="formula" title='Vector riêng ψₖ (eq 28)' %}

Từ hàng đầu tiên của \(Q(x_k)\boldsymbol{\psi}_k = \mathbf{0}\):

\[ \boldsymbol{\psi}_k = \begin{pmatrix} \mu_1 x_k \\ (\lambda + \mu_1)x_k - \lambda x_k^2 \end{pmatrix} = \begin{pmatrix} \mu_1 x_k \\ x_k[(\lambda + \mu_1) - \lambda x_k] \end{pmatrix} \]

Đặc biệt với \(r_1 = 0\): \(\boldsymbol{\psi}_1 = (0, 0)\) — nghiệm tầm thường, cần xử lý riêng.

{% include components/box-end.html %}

6. Hệ số a₁, a₂, a₃ (Coefficients — Proposition 3)

Xác suất trạng thái dừng được biểu diễn:

\[ \boldsymbol{\pi}_i = a_1 r_1^i \boldsymbol{\psi}_1 + a_3 r_3^i \boldsymbol{\psi}_3 + a_4 r_4^i \boldsymbol{\psi}_4, \quad i \geq 1 \]

Vì \(r_1 = 0\), số hạng đầu tiên chỉ đóng góp tại \(i = 1\) (theo quy ước \(0^0 = 1\)), nên thực tế ta cần xác định 3 hệ số \(a_1, a_3, a_4\) từ:

{% include components/box-start.html type="theorem" title='Proposition 3 (eqs 29–31)' %}

Ba hệ số \(a_1, a_3, a_4\) được xác định bởi hệ phương trình tuyến tính từ điều kiện biên (phương trình cân bằng tại mức 0 và 1) và điều kiện chuẩn hóa:

  1. Phương trình cân bằng tại \((1,1)\) và \((1,2)\).
  2. Điều kiện chuẩn hóa: \(\pi_{0,0} + \sum_{i=1}^{\infty}(\pi_{i,1} + \pi_{i,2}) = 1\).

Hệ này có nghiệm duy nhất vì các trị riêng phân biệt và nằm trong đường tròn đơn vị.

{% include components/box-end.html %}

7. Xác suất trạng thái dừng (Steady-State Probabilities)

{% include components/box-start.html type="definition" title='Xác suất trạng thái dừng (eqs 32–34)' %}

Sau khi giải hệ phương trình, các xác suất trạng thái dừng quan trọng:

Xác suất hệ trống:

\[ \pi_{0,0} = \frac{\mu_2}{\lambda} \cdot (\pi_{1,2}) \tag{32} \]

Xác suất tại mức 1:

\[ \pi_{1,1} = a_1 \psi_{1,1} + a_3 r_3 \psi_{3,1} + a_4 r_4 \psi_{4,1} \tag{33} \] \[ \pi_{1,2} = a_1 \psi_{1,2} + a_3 r_3 \psi_{3,2} + a_4 r_4 \psi_{4,2} \tag{34} \]

Xác suất tại mức \(i \geq 2\):

\[ \pi_{i,j} = a_3 r_3^i \psi_{3,j} + a_4 r_4^i \psi_{4,j}, \quad j \in \{1, 2\} \] {% include components/box-end.html %}

8. Công thức hiệu năng (Performance Measures)

{% include components/box-start.html type="formula" title='Các chỉ số hiệu năng (eqs 35–37)' %}

Số khách trung bình trong hệ \(L\):

\[ L = \sum_{i=1}^{\infty} i \cdot (\pi_{i,1} + \pi_{i,2}) = \sum_{k \in \{3,4\}} a_k \frac{r_k}{(1-r_k)^2}(\psi_{k,1} + \psi_{k,2}) + (\pi_{1,1} + \pi_{1,2}) \tag{35} \]

Thời gian chờ trung bình \(W\) (theo Little's law):

\[ W = \frac{L}{\lambda} \tag{36} \]

Xác suất bị chặn \(P_B\):

\[ P_B = \sum_{i=1}^{\infty} \pi_{i,2} = \sum_{k \in \{3,4\}} a_k \frac{r_k}{1-r_k} \psi_{k,2} + \pi_{1,2} \tag{37} \]

Đây là xác suất server 2 đang bận — khi một khách mới hoàn thành tại server 1, nó sẽ bị chặn với xác suất này.

{% include components/box-end.html %} {% include components/box-start.html type="summary" title='Tóm tắt chương' %}
  • Phương trình \(\det[Q(x)] = 0\) là đa thức bậc 4 — giải tường minh bằng quartic formula.
  • 3 trị riêng nằm trong đường tròn đơn vị: \(r_1 = 0, r_3, r_4\).
  • Vector riêng và hệ số xác định duy nhất từ điều kiện biên + chuẩn hóa.
  • Xác suất dừng có dạng đóng: \(\pi_{i,j} = \sum_k a_k r_k^i \psi_{k,j}\).
  • Chỉ số hiệu năng \(L, W, P_B\) đều có công thức tường minh.
